# docs/

Coloca aquí los recursos visuales del README:

- **`preview.png`** — captura de la ventana de la herramienta (ya la tienes).
- **`demo.gif`** — (opcional pero recomendado) un GIF corto mostrando el flujo:
  seleccionar objetos → Smart Rename → preview → Aplicar → log.

El README los referencia como `docs/preview.png` y `docs/demo.gif`.
Una vez que agregues los archivos, puedes borrar este `.md`.

## Cómo grabar el GIF (sin instalar nada raro)
- Windows: **Xbox Game Bar** (`Win + G`) para grabar, luego convertí el `.mp4`
  a `.gif` en https://ezgif.com/video-to-gif (es lo que usaste antes).
- O usá **ScreenToGif** (gratis, liviano) para grabar directo a GIF.
